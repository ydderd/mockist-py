# mockist-py

Python port of [mockist](https://github.com/ydderd/mockist) — a local test harness for
agent tool/skill boundaries: stub whole tool calls, record/replay cassettes, and assert
trajectories.

**Status:** early — implementation spec only. See [docs/SPEC.md](docs/SPEC.md).

The TypeScript package (`@ydderd/mockist`) is the shipping reference implementation and
cassette format v1 authority: https://github.com/ydderd/mockist
